# LOGOSBOOK
Book-reading app with AI features

# Installation

## Stack
Python, NodeJS, VueJS

## Clone repo
`git clone --recurse-submodules https://github.com/NightyStudios/Logosbook_Back`

## Install necessary libraries
`pip install -r requirements.txt`

## Run BackEnd
`uvicorn main:app --reload`

## Setup FrontEnd

```bash
cd Logosbook_Front
npm install
```
## Run FrontEnd
`npm run dev`

## Enjoy!
